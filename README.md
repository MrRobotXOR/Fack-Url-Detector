# Fake URL Detector

Fake URL Detector is a web-based tool that helps identify fake and suspicious URLs.

## Features
- Detects fake and phishing URLs
- Supports `http://` and `https://` URLs
- Fast and simple URL checking

## Important Note
⚠️ This project currently works correctly only with URLs that include:

- `http://`
- `https://`

### Supported URL Format
```txt
https://google.com
http://example.com

## Unsupported URL Format
www.google.com
google.com
